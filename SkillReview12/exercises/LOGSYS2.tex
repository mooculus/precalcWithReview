\begin{exercise}
Solve the following system of equations. Hint: You might need to use log rules.\\
\[
\begin{cases}
\ln(xy) = 5\\
\ln\left(\frac{x}{y}\right) = 3
\end{cases}
\]
\[
x = \answer{e^4} y = \answer{e}
\]

\end{exercise}
