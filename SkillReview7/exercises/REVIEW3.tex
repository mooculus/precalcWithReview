\begin{exercise}
Write the result of the operation on the function $f(x)$.\\
$f(x)=x^2$\\
$2f(x) = \answer{2x^2}$\\
$f(2x) = \answer{4x^2}$ \\
$f(x-3) = \answer{(x-3)^2}$ \\
$f(x)-3 = \answer{x^2-3}$\\
$2f(x)-3 = \answer{2x^2-3}$
\end{exercise}
