\begin{exercise}
Write the result of the operation on the function $f(x)$.\\
$f(x)=\ln(x)$\\
$2f(x) = \answer{2\ln(x)}$\\
$f(2x) = \answer{\ln(2x)}$ \\
$f(x-3) = \answer{\ln(x-3)}$ \\
$f(x)-3 = \answer{\ln(x)-3}$\\
$2f(x)-3 = \answer{2\ln(x)-3}$
\end{exercise}
