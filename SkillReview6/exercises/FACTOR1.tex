\begin{exercise}
Use factoring to solve the equation.\\
$x^4-8x^2+16 =0$\\
$x^4-8x^2+16 = (\answer{x^2-4})^2=0$\\
Larger value of $x=\answer{2}$
Smaller value of $x=\answer{-2}$
\end{exercise}
