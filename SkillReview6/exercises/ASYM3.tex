Given the following rational function $f(x)$, answer the question below.
\begin{exercise}
\[
f(x)=\frac{x}{x^2-2x-8}
\]
What can we say about the function when $x=0$?
\begin{multipleChoice}  
\choice{There is a hole at $x=0$.}  
\choice{There is a vertical asymptote at $x=0$.}  
\choice[correct]{There is a defined value at $x=0$.}
\choice{There is a horizontal asymptote at $x=0$.}  
\end{multipleChoice}  
\end{exercise}
