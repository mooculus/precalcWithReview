Given the following rational function $f(x)$, answer the question below.
\begin{exercise}
\[
f(x)=\frac{1}{x+3}
\]
What can we say about the function when $x=-3$?
\begin{multipleChoice}  
\choice{There is a hole at $x=-3$.}  
\choice[correct]{There is a vertical asymptote at $x=-3$.}  
\choice{There is a defined value at $x=-3$.}
\choice{There is a horizontal asymptote at $x=-3$.}  
\end{multipleChoice}  
\end{exercise}
