A human running a $40$ yard dash in $4$ seconds is considered extremely fast. Complete the following problems to figure out how fast that is in miles per hour.
\begin{exercise}
How many yards would be run in $1$ second?\\
$\answer{10}$ yards in $1$ second.
\end{exercise}
\begin{exercise}
How many miles would be run in $1$ second? There are $1760$ yards in $1$ mile. Leave the answer as a calculator ready response not a decimal approximation.\\
$\answer{\frac{10}{1760}}$ miles
\end{exercise}
\begin{exercise}
How many seconds are in $1$ hour?\\
$\answer{3600}$ seconds
\end{exercise}
\begin{exercise}
Type a calculator ready response to determine miles per hour and the approximate answer rounded to 2 decimal places.\\
Calculator ready: $\answer{\frac{36000}{1760}}$\\
Decimal approximation: $\answer{20.45}$ \calcHW miles per hour
\end{exercise}
